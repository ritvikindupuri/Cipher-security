## SIEM/Log Query Examples

For each query, provide ONE separate code block. Label each with the platform it runs on:

### Splunk (SPL)
```splunk

```

### Microsoft Sentinel (KQL)
```kql
SecurityEvent | where Process == "suspicious.exe"
```

### Elastic (Lucene)
```lucene
process.name:suspicious.exe AND event.code:4688
```

### GrayLog / QRadar
```sql
SELECT * FROM windows_events WHERE process_name = 'suspicious.exe'
```
