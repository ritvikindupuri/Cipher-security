## YOUR TASK

- If data is missing, state "No data available"  
- Be technically precise with numbers, IPs, process IDs
- Use security terminology appropriate for SOC analysts and incident responders

## Telemetry Data Reference

The telemetry JSON includes:
- `snapshot.host` - System hostname and OS info
- `snapshot.cpu.usage_percent_interval_0_5s` - CPU usage percentage
- `snapshot.memory.virtual` - Virtual memory usage
- `snapshot.disks` - Disk usage and information
- `snapshot.processes` - Running processes with resource usage
- `snapshot.network_connections` - Network connections and ports
- `snapshot.network_io` - Network input/output statistics

Provide detailed, actionable findings - this is for security professionals.
